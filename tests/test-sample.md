## Try It Out!

1. Click the formatting buttons in the toolbar above
2. Use keyboard shortcuts:
   - `Ctrl+B` for **bold**
   - `Ctrl+I` for *italic*
   - `Ctrl+K` for links
   - `Ctrl+S` to save
3. Switch between view modes using the dropdown
4. Export to HTML or PDF when you're done
