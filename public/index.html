Quick Installation

1

Download the Extension

Download the latest .vsix file from the GitHub Releases page

2

Install in VS Code

Install the extension using the command line or VS Code:

code --install-extension md-office-editor-0.2.1.vsix

Or in VS Code: Extensions view ‚Üí "..." menu ‚Üí "Install from VSIX..."

3

Start editing

Open any .md file, right-click and select "Open with MDOffice - Markdown Office Editor" to start using the extension.
